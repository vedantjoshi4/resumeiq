# ResumeIQ — AI Resume Analyzer

An AI-powered tool that analyzes your resume against any job description and gives instant feedback.

🔗 Live Demo: https://vedantjoshi4.github.io/resumeiq

## What it does
- Gives ATS match score out of 100
- Shows keywords present vs missing
- Points out strengths and weaknesses
- Gives specific rewrite suggestions

## Tech used
- HTML, CSS, Vanilla JavaScript
- Claude AI API by Anthropic

## How to use
1. Get a free API key from https://console.anthropic.com
2. Open the live demo link above
3. Paste your resume text
4. Paste the job description
5. Click Analyze — get results in seconds
